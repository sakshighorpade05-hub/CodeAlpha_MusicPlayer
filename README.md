# CodeAlpha Music Player

A responsive and interactive **Music Player** built using **HTML**, **CSS**, and **JavaScript** as part of the **CodeAlpha Frontend Development Internship**.

##  Features

-  Play and Pause music
-  Previous and ⏭️ Next song controls
-  Album cover display
-  Interactive progress bar
-  Current time and total duration
-  Volume control
-  Playlist support
-  Auto play next song
-  Fully responsive design

##  Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
